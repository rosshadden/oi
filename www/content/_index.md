Oi is a general purpose system language with a high emphasis on ergonomics.

It is written for humans, by human([s?](https://github.com/rawsp33d/oi)).
More specifically, it's for humans who love programming.

Oi is designed such that the code you want to write is usually the code you actually write.
Where other languages optimize for things like safety, perf, or simplicity, Oi optimizes for flow.

Its features encourage uninterrupted thought:
- implicit returns
- emphasis on expressions
- leading literals
- trailing functions
- pipelines
- named returns
- zero-value initialization
